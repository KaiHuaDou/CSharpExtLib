标准 C# 库扩展

> Author: KaiHuaDou

## 功能
> Version 1.4.5

### CSharpExtLib
+ SystemManagement: 系统管理相关工具

### CSharpExtLib.Extensions
+ 各个 .NET 内置类型的扩展类

### CSharpExtLib.Char
+ FontManager：一键判断字体类型是否为符号字体（如 Wingdings）
+ JumpChar：控制台实现文本"打字机"效果

### CSharpExtLib.Easy
+ EasyColorMix：模拟 Photoshop 的各种颜色混合模式的实用类
+ EasyList：使用 XML 储存列表对象的实用类
+ EasyConvert：转换 WinForm 和 WPF 数据结构的静态实用类

### CSharpExtLib.Math
+ Fraction：进行分数运算的实用类
+ Universal：通用数学运算

### CSharpExtLib.WPF
+ ColorPicker：WPF 颜色选择器
+ FontSubDialog: WPF 字体选择器
+ FontDialog：**WPF 原生**字体选择器
