# CSharpExtLib
一些十分有用的C#代码段
> Author: KaiHuaDou

# 目前功能

## CSharpExtLib
+ FontManager：一键判断字体类型是否为符号字体（如Wingdings）
+ JumpChar：控制台实现文本“打字机”效果

## CSharpExtLib.Easy
+ EasyColor：模拟Photoshop的各种颜色混合模式的实用类。
+ EasyConfig：使用XML序列化对象（通常是设置）的实用类。
+ EasyConverter：转换WinForm和WPF数据结构的静态实用类

## CSharpExtLib.Math
+ Fraction：进行分数运算的实用类
+ StdApi：其他数学运算
