# CSharpExtLib
一些十分有用的C#代码段
> Author: KaiHuaDou

# 目前功能
> Version 1.3

## CSharpExtLib
+ Universal：通用静态方法集。

## CSharpExtLib.Char
+ FontManager：一键判断字体类型是否为符号字体（如 Wingdings）。
+ JumpChar：控制台实现文本"打字机"效果。

## CSharpExtLib.Easy
+ EasyColorMix：模拟 Photoshop 的各种颜色混合模式的实用类。
+ EasyConfig：使用 XML 序列化对象（通常是设置）的实用类。
  + EasyConfigList：管理多个对象的实用类。
  + EasyConfigOne： 管理一个对象的实用类。
+ EasyConvert：转换 WinForm 和 WPF 数据结构的静态实用类

## CSharpExtLib.Math
+ Fraction：进行分数运算的实用类。
+ StdApi：其他数学运算。

## CSharpExtLib.WPF
+ ColorPicker：WPF 颜色选择器。
+ *(Beta) FontDialog：WPF 字体选择器*
